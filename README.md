# AI Startup Branding Generator

## Overview

AI Startup Branding Generator is a web application that generates branding ideas for startup businesses based on user input.

The application generates:

- Startup Name
- Tagline
- Brand Colors
- Logo Idea
- Target Audience
- Marketing Strategy

---

## Features

- User-friendly interface
- Dynamic startup branding generation
- Responsive design
- FastAPI backend
- HTML, CSS, JavaScript frontend
- Offline branding logic (no API required)

---

## Technologies Used

- Python
- FastAPI
- HTML5
- CSS3
- JavaScript

---

## Project Structure

```
Backend/
    app.py
    generator.py

Frontend/
    index.html
    style.css
    script.js
```

---

## How to Run

1. Clone the repository

```
git clone https://github.com/YourUsername/AI-Startup-Branding-Generator.git
```

2. Open the Backend folder

```
cd Backend
```

3. Install dependencies

```
pip install fastapi uvicorn
```

4. Start the backend

```
python -m uvicorn app:app --reload
```

5. Open `index.html` using Live Server.
